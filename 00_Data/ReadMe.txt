The following data sets are Used

    1.  kontas_2017 (DOI: https://doi.org/10.5281/zenodo.16647156)
        downloaded at 23/01/2026 from https://zenodo.org/records/16647156
        Content: 770 all-sky images with masks (day time obversations only)

    2.  ASI (DOI: https://doi.org/10.5281/zenodo.12804613)
        downloaded at 10/06/2026 from https://eye2sky.de/data/asi/imgs/
        Content: all-sky images of several stations in Germany but for this
            the data set was reduced to:
            All-sky images from station AURIC between 01/04/2022-01/01/2023
            with a time resolution of 5 minutes 
            -> 4986 all-sky images (with time stamps)
        For more insights: https://eye2sky.de/data/data_description/
